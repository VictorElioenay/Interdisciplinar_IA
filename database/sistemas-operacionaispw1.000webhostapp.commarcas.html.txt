                Windows 95
                O Microsoft Windows 95 (codinome Chicago)  um sistema operacional de 16/32 bits criado pela empresa
                    Microsoft. Lanado em 24 de agosto de 1995, o Windows 95 revolucionou o mercado de sistemas
                    operacionais e passou a vir instalado por padro com o MS-DOS 7.0 (e no mais separado, como era
                    antes), sendo o principal lanamento da empresa na dcada de 1990. Entre outras coisas, efetivava o
                    sistema de arquivos FAT-16 (ou VFAT). Os ficheiros (arquivos) puderam a partir de ento ter 256
                    caracteres e no apenas 8 como sucedia nas verses anteriores. Os nomes longos passam a ser
                    armazenados em uma tabela aparte, mantendo assim a compatibilidade com o MS-DOS. O salto do Windows
                    3.1 para o Windows 95 foi enorme e fez com que a Microsoft pulasse para ser a mais popular
                    distribuidora de sistemas operacionais.
                Um dos grandes problemas do Windows 95 era a instabilidade ao rodar aplicativos escritos para o
                    Windows 3.x, pois o Windows 95 passava a executar a multitarefa cooperativa (exatamente como o
                    Windows 3.x) entregando ao aplicativo o gerenciamento do processador e da memria, podendo resultar
                    em travamentos e reinicializaes do computador, caso o aplicativo executasse alguma tarefa
                    indevida. Portanto, para evitar travamentos no Windows 95 (para quem ainda o usa),  bom evitar usar
                    aplicativos escritos para o Windows 3.x.
                Existe uma outra verso do Windows 95, lanada em 1996, chamada de Windows 95 OEM Service Release 2
                    (OSR 2), com suporte nativo ao sistema de arquivos FAT32. J o Windows 95, a partir da reviso OSR
                    2.1, inclua o suporte nativo ao Barramento Serial Universal e ao Ultra DMA (veja tabela abaixo).
                    Teve o suporte encerrado em 2002.
